        \frametitle{Entering citations in slides}
        You can quote articles books and and other references from your bibtex file. A possible quote for an artice would look like this \cite[Non-cooperative Games]{nash51}.
        
        If you do not enter a short description a citation for a book looks like this \cite{hawking1988}.

        A link to a website can be referenced in bibtex with the misc flag see for example \cite{CitekeyMisc}.
