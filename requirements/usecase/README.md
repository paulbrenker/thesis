# Use Cases

Use cases in software development represent scenarios or sequences of actions that convey how a system interacts with users or other systems to achieve a specific function or task.

## Template

### Use Case Name

- **System:** _System Name_
- **Use Case:** _Use Case Number_
- **Akteur:** _Actors_
- **Flow:** _Flow of Actions taken in the Scope of the Use Case._
- **Nachbedingung:** _Conditions met after execution of Use Case_
- **Qualitätsanforderungen:**
  - _List of quality requirements the use case has to fulfill (must be mapped to software quality attributes)_

In addition to the textual use case a UML Use Case Diagram can be supplied.
